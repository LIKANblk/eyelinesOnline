Чтобы построить all_tables я использовала функцию results_for_tables(тут номера экспериментов) из пакета eyelinesOnline, пример:

 results_for_tables(c(29,30,32,33,34,35,36,37,38))

эта функция сохраняет на яндекс-диск в папку eyelinesOnlineNew/tables три файла:

moves_table.csv - Таблица с данными по фиксациям в разных частях эксперимента
report_summary_no_change.csv - Таблица с выходом классификатора
results_clf.csv - Таблица с сырыми результатами классификации

Чтобы сделать summary_new_task, я запускаю randov_vs_normal_clf_compare_tables(тут номера экспериментов). Эта функция тоже сохраняет csv в папку eyelinesOnlineNew/tables 4 файла:
df_test_ball
df_test_field
df_random_ball
df_random_field
там таблицы, где ячейки с суффиксом, соответствующим типу задания - rem - remembering task (задание на запоминание), seq - sequence task (задание на выкладывание последовательности)


перезапуск демона ядиска можно сделать через ssh, нужно зайти с логином root (пароль gentoo) и выполнить 
/etc/init.d/supervisord restart

Чтобы сгенерировать mat файлы, сначала нужно, чтобы в папке был файл experiment.RData, он делается с помощью функции process_experiment('path')

После этого нужно запустить функцию generate_files_for_matlab('path').  Она достаточно долго работает (около 3 минут на один эксперимент).

- нужно писать путь к файлу experiment.RData, пример:

generate_files_for_matlab('~/Yandex.Disk/eyelinesOnlineNew/data/28/experiment.RData')
